本文档注释：
data：实验过程中产生的所有数据，具体信息可见具体文件夹
chem.py:从mark中检索固定药物的信息语句；
grap.py:将mark中的关键词关系读取，并储存为sif的格式，从而进行关系网络图的绘制；
PMID.py:将通过esearch抓取的文献的pmid单独提取，从而进行进行pubtator的抓取；
pubtator.py:根据pubtator的数据，抽取出现超过10次的生物学名词以及频率存入相应文件中；
relation.py:将shai2中的句子筛选出处于主宾节点的关键词读取，以序号，类型，关键词的格式输出mark中
rode.py:对mark中的成对关系进行依存路径分析，并找出关键词的最短路径；
shai.py:具体筛选方法，从所有句子中，根据预定的条件，筛选出满足条件的句子；
